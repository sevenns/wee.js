### - Class

- **addClass:**
```javascript
  $('ul').addClass('class1 class2');
```

- **removeClass:**
```javascript
  $('ul').removeClass('class1 class2');
```

- **toggleClass:**
```javascript
	$('ul').toggleClass('class1 class2');
```

- **hasClass:**
```javascript
	$('ul').hasClass('class');
```
