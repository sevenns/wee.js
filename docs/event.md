### - Events

- **ready:**
```javascript
  // Document ready
  $.ready(function () {
    // do something
  });
```

- **resize:**
```javascript
  // Window resize
  $.resize(function () {
    // do something
  });
```

- **scroll:**
```javascript
  // Document scroll
  $.scroll(function () {
    // do something
  });
```

- **load:**
```javascript
  // Window load
  $.load(function () {
    // do something
  });
```

- **on:**
```javascript
  $(selector).on('click', function () {
    // do something
  });
```
